# Job Market Analysis using Web Scraping & EDA

This project focuses on analyzing the current job market by collecting job listings data through web scraping using Selenium and performing Exploratory Data Analysis (EDA) using Python libraries like Pandas, Matplotlib, and Seaborn.

The dataset was scraped from job portals and includes information such as:

* Job Title
* Company Name
* Location
* Experience Required
* Skills Required

## Project Objectives

* Scrape real-time job market data using Selenium
* Create a structured dataset using Pandas DataFrame
* Clean and preprocess the collected data
* Perform Exploratory Data Analysis (EDA)
* Visualize hiring trends, top skills, locations, and companies

## Technologies Used

* Python
* Selenium
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

## Key Insights

* Identified the most in-demand skills in the job market
* Analyzed top hiring companies
* Found locations with the highest number of job openings
* Explored experience-level requirements across jobs

## Visualizations Included

* Top Hiring Companies
* Top Job Locations
* Most Demanded Skills
* Experience Distribution
* Skills Word Cloud
* Missing Values Heatmap
